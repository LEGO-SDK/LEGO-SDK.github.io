FileManager

FileManager

使用 FileManager 操控沙箱中的任意文件

你可以操控 Document / Caches / Tmp 目录中的任何文件，不受任何限制，但请注意以下要点。

  • Document 该目录只能存放由用户主动生成的数据，不能存放缓存数据
  • Caches 该目录存放缓存数据
  • Tmp 该目录下的文件，会随时被清除

模块提供四个操作方法，写入、读取、检查、删除。

  • read = 读取文件
  • update = 写入文件
  • delete = 删除文件
  • check = 检查文件是否存在

Interface


kFileSuiteCache = "Cache"
kFileSuiteDocument = "Document"
kFileSuiteTmp = "Tmp"

lego.native.file.exist: (filePath: string, suite?: string) => Promise<boolean>
lego.native.file.read: (filePath: string, suite?: string) => Promise<string>
lego.native.file.write: (filePath: string, fileContents: string, suite?: string) => Promise<void>
lego.native.file.remove: (filePath: string, suite?: string) => Promise<void>

Example

lego.native.file.read("test.txt", kFileSuiteCache)
    .then(function(fileContent){
        // do something    
    })

lego.native.file.write("test.txt", "hello world", kFileSuiteCache)
