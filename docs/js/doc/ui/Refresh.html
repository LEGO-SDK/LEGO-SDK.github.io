Refresh

Refresh

使用 Refresh 模块可以为 WebView 添加一个原生的下拉刷新控制器。

Interface

lego.ui.refresh.enable: (callback?: () => void) => void
lego.ui.refresh.complete: () => void

Example

// 刷新开启，5s后关闭
lego.ui.refresh.enable(function(){
    setTimeout(function(){
        lego.ui.refresh.complete()
    }, 5000)
})


// 下拉拉取数据，完成后关闭
lego.ui.refresh.enable(function(){
    lego.native.request("https://status.github.com/api.json")
        .then(function(result){
            // ...
            lego.ui.refresh.complete()
        })
})
