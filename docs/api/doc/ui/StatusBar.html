StatusBar

StatusBar

使用 StatusBar 模块可以修改状态栏样式。

RequestParams

STYLE style;
Bool  hidden;
Bool  animated;

enum STYLE {
    case "default"
    case "light"
}

ResponseParams

Example

JSMessage.newMessage("UI.StatusBar", {
    style: 'light',
    hidden: false,
    animated: true,
}).call(null)
