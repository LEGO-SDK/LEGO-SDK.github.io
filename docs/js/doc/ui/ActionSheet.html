ActionSheet

ActionSheet

使用 ActionSheet 可以弹出一个包含多个选项的对话框，用户选择对应选项后会有回调。点击取消选项和点击蒙版区域触发相同的事件。

Interface

lego.ui.sheet: (
    title: string, 
    choices: string[], 
    cancelText?: string, 
    redIndex?: number) => Promise<number>

Example

lego.ui.sheet("请问，罗老板隐婚了多久？", ["一年", "两年", "三年", "五年"], "我选择死亡", 3)
    .then(function(tapIndex){
        console.log(tapIndex)
    })

// 不指定 cancelText 和 redIndex
lego.ui.sheet("请问，罗老板隐婚了多久？", ["一年", "两年", "三年", "五年"])
    .then(function(tapIndex){
        console.log(tapIndex)
    })

// 仅指定 redIndex 
lego.ui.sheet("请问，罗老板隐婚了多久？", ["一年", "两年", "三年", "五年"], undefined, 3)
    .then(function(tapIndex){
        console.log(tapIndex)
    })
