LEGO SDK

轻量、易用的 WebView 增强组件

开始使用

轻量

核心库不超过十个类，扩展模块相互分离。

通用

跨平台，一致的 API 封装以及 API 调用方法。

易用

iOS 只需添加 Pods 依赖，Android 只需配置几行 Gradle 代码。

扩展

数十个官方 API 可供使用，开发者自行扩展亦相当简单。
