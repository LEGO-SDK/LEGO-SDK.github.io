LEGO-SDK

NavigationBar

使用 NavigationBar 模块可以修改状态栏样式。

RequestParams

Bool hidden;
Bool animated;

ResponseParams

Example

JSMessage.newMessage("UI.NavigationBar", {
    hidden: true,
    animated: true,
}).call(null)
