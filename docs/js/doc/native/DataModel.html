DataModel

DataModel是原生应用与WebView进行数据交换的桥梁。

  • 原生应用可以通过 DataModel 模块，向 WebView 传递数据。
  • WebView 也可以通过 DataModel 模块，向 原生应用 传递数据。

Interface

lego.native.data: (key: string, value?: any) => Promise<any>

Example

// 读取
lego.native.data.get("aKey")
    .then(function(value){
        // ...
    })

// 设置
lego.native.data.set("aKey", "aValue")

// 监听
lego.native.data.watch("aKey", function(newValue){

})
